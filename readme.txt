== Description ==

**Database Analyzer** is a powerful tool for WordPress website owners to optimize, clean, and improve the performance of their MySQL database. This plugin provides an easy way to analyze database tables, clean up unnecessary data — all essential steps to ensure your WordPress site runs smoothly and performs at its best.

Key features included:

– **Analyze Database**: View detailed information about your database tables, including the number of rows, table sizes.
– **Optimize Tables**: Automatically optimize your database tables to improve query performance.
– **Cleanup Overhead**: Remove unnecessary data, including post revisions, trashed items, and transient options.
– **Easy-to-Use Interface**: A simple, user-friendly interface that allows you to run database optimization with a click of a button.
– **Database Insights**: Get clear insights into how your database is performing and identify areas for improvement.

**Benefits:**
– Faster website performance by reducing database bloat.
– Improved page load times.
– Cleaner and more efficient database.

== Installation ==

1. Upload the `database-analyzer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'DB Analyzer' menu in your WordPress dashboard to start optimizing your database.
4. Use the available options to analyze, optimize, and clean up your database.
